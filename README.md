# Final Portfolio

This version follows the final direction exactly:

- dark professional theme
- simple hero
- photo right-aligned on desktop
- no CV section
- no download CV button
- no YN logo/brand in the top left
- no class year badge under the photo
- simple section headings only
- read-only public portfolio
- easy-to-edit structure

## Folder structure

```text
final_portfolio/
├── index.html
├── style.css
├── script.js
├── assets/
│   └── notes.txt
└── README.md
```

## What to edit

Most of your changes will be in **index.html** only.

### 1. Hero
Replace:
- `Your Name`
- photo file at `assets/photo.jpg`

### 2. About
Edit the two paragraphs inside the `ABOUT` section.

### 3. Highlights
Copy one highlight card block to add more.

### 4. Certifications
Each certificate card has:
- provider name
- certificate name
- image path

To add more certificates:
- copy one full `cert-card`
- paste it below the others
- change the text
- add the scanned image to `assets/`

### 5. Honors & Awards
Use this for:
- scholarships
- awards
- recognitions
- merit-based achievements
- competition wins

To add more items:
- copy one `timeline-item`
- paste it below
- change the year and text

### 6. Education & Experience
Use this for:
- your degree / education story
- internships
- field / attachment experience
- major growth milestones

### 7. Contact
Replace the placeholder email, LinkedIn, GitHub, and WhatsApp links.

### 8. Footer
Replace `Your Name` in:

```text
© 2026 Your Name. All rights reserved.
```

## Assets
Add your files into the `assets` folder.

Recommended names:
- `photo.jpg`
- `certificate-1.jpg`
- `certificate-2.jpg`
- more certificate images as needed

## GitHub Pages

1. Upload all files to your repository
2. Go to **Settings → Pages**
3. Choose:
   - **Deploy from a branch**
   - branch: `main`
   - folder: `/ (root)`
4. Save

## Editing tip

To keep things easy, avoid changing `style.css` unless you want design changes.
For normal updates, edit only:
- `index.html`
- image files in `assets/`
